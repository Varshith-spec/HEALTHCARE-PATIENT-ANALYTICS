# HEALTHCARE-PATIENT-ANALYTICS
# Healthcare Patient Analytics & Hospital Operations Dashboard

## Project Overview
This project analyzes hospital patient data to identify operational inefficiencies and improve healthcare performance using Python, SQL, and Power BI.

## Business Problem
Hospitals often face issues such as:
- Long patient wait times
- High readmission rates
- Poor bed utilization
- Lack of operational visibility

## Tools Used
- Python (Pandas, Matplotlib)
- SQL
- Power BI
- Excel

## Key KPIs
- Total Patients
- Average Wait Time
- Readmission Rate
- Bed Occupancy
- Length of Stay

## Dashboard Insights
- Emergency department has the highest wait times
- Certain departments have higher readmission rates
- ICU beds show the highest occupancy
